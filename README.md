## Setup
1. Fork/upload repo ini ke GitHub
2. Connect ke Vercel
3. Isi Environment Variables di Vercel:

| Variable | Dari mana | Wajib? |
|----------|-----------|--------|
| `SUPABASE_URL` | supabase.com → Settings → API | ✅ Wajib |
| `SUPABASE_KEY` | supabase.com → Settings → API (anon public) | ✅ Wajib |
| `GROQ_KEY` | console.groq.com | ✅ Wajib |
| `GEMINI_KEY` | aistudio.google.com | Opsional |
| `OPENROUTER_KEY` | openrouter.ai | Opsional |

4. Deploy → selesai!
